# Task Tracker

A browser-based Task Tracker application built using HTML, CSS, JavaScript, and TypeScript.

## Features

- Add new tasks
- Mark tasks as Done/Pending
- Filter tasks (All, Pending, Done)
- Sort tasks by Priority or Due Date
- Store tasks using Local Storage
- Summary table
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- TypeScript
